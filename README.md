## Usage
To run the BitTorrent client in TUI mode: 
`cargo run`


| Key | Action |
| -------- | ------- |
| `q`  | Quit out of TUI |
| `d` or `1` | Navigate to Downloads Tab |
| `t` or `2` | Navigate to Torrent Tab |
| `l` or `3` | Navigate to Logs Tab |
